# IDENTITY ROLE ENGINE

This prompt must be executed using the Whycespace MASTER PROMPT GENERATION TEMPLATE.

This component implements the Identity Role Engine responsible for managing role assignments for identities.

Roles define privileges and responsibilities across the Whycespace ecosystem.

The engine performs controlled mutation of identity roles.

All architecture rules defined in:

claude/bootstrap-prompts/master-bootstrap.prompt.md

must be respected.

SECTION 2 — ARCHITECTURE VERIFICATION
------------------------------------------------------------

Identity Role Engine performs controlled mutation of identity roles.

Responsibilities:

• assign roles to identities
• revoke roles from identities
• validate role assignments
• emit role change events

The engine must NOT:

• persist data
• call other engines
• access infrastructure directly

Persistence occurs through system stores.
