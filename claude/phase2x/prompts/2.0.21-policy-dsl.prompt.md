SECTION 2 — COMPONENT PURPOSE
------------------------------------------------------------

The Policy DSL defines the canonical **policy definition language** used by the WhycePolicy system.

It provides the schema used to describe governance policies including:

• policy identity
• policy domain
• policy conditions
• policy actions
• policy priorities
• policy lifecycle

The DSL does not execute policies.

Policy evaluation is performed by the **Policy Evaluation Engine**.

This component defines **policy structure only**.
