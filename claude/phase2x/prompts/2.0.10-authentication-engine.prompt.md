# AUTHENTICATION ENGINE

This prompt must be executed using the Whycespace MASTER PROMPT GENERATION TEMPLATE.

This component implements the Authentication Engine responsible for verifying identity credentials and producing authentication decisions.

Authentication determines whether an identity can successfully establish an authenticated session within the Whycespace ecosystem.

Authentication evaluates:

identity existence
identity status
credential validation
device trust evaluation
verification requirements

All architecture rules defined in:

claude/bootstrap-prompts/master-bootstrap.prompt.md

must be respected.

SECTION 2 — ARCHITECTURE VERIFICATION
------------------------------------------------------------

Authentication Engine performs operational authentication validation.

Responsibilities:

• validate identity credentials
• evaluate identity status
• verify authentication factors
• evaluate device trust score
• produce authentication decision

Authentication Engine must NOT:

• persist identity data
• modify identity models
• access infrastructure
• directly create sessions

It produces authentication decisions only.
