# SESSION ENGINE

This prompt must be executed using the Whycespace MASTER PROMPT GENERATION TEMPLATE.

This component implements the Session Engine responsible for managing authenticated session lifecycle within the Whycespace ecosystem.

Sessions represent authenticated identity access contexts used by services, APIs, and workflows.

Session evaluation manages:

session creation
session validation
session expiration
session revocation

All architecture rules defined in:

claude/bootstrap-prompts/master-bootstrap.prompt.md

must be respected.

SECTION 2 — ARCHITECTURE VERIFICATION
------------------------------------------------------------

Session Engine performs session lifecycle operations.

Responsibilities:

• create authenticated session
• validate session state
• evaluate session expiration
• revoke active sessions
• generate session tokens

Session Engine must NOT:

• mutate identity state
• access infrastructure directly
• persist external data
• call external services

It produces session lifecycle decisions only.
