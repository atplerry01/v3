SECTION 1 — ARCHITECTURE VERIFICATION
------------------------------------------------------------

You are implementing the **Integrity Verification Engine** for the WhyceChain system.

WhyceChain provides the immutable evidence infrastructure for Whycespace governance.

The Integrity Verification Engine validates the **cryptographic integrity of the ledger and blocks**.

The engine ensures that:

• ledger entries are not tampered with
• entry hash chain is valid
• blocks link correctly
• Merkle roots match entries
• Merkle proofs are valid

The engine produces **verification reports** but does not mutate system state.

------------------------------------------------------------

Ledger Integrity

• sequence continuity
• previous entry hash linkage
• entry hash validity

Block Chain Integrity

• block height continuity
• previous block hash linkage
