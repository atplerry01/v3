# 2.0.50 — Chain Index Engine

## Component

- **Number:** 2.0.50
- **Name:** Chain Index Engine
- **System:** WhyceChain System
- **Layer:** Engines Layer
- **Tier:** T3I — Intelligence Engine

## Description

Builds deterministic indexes for WhyceChain ledger entries and blocks.
Indexes allow efficient retrieval by sequence, hash, height, trace ID, and correlation ID.

## Indexes Produced

| Index | Key | Value |
|-------|-----|-------|
| EntryIndexBySequence | sequence (long) | entryHash (string) |
| EntryIndexByHash | entryHash (string) | sequence (long) |
| BlockIndexByHeight | height (long) | blockHash (string) |
| BlockIndexByHash | blockHash (string) | height (long) |
| TraceIndex | traceId (string) | evidence hashes (List) |
| CorrelationIndex | correlationId (string) | evidence hashes (List) |
