------------------------------------------------------------
SYSTEM CONTEXT
------------------------------------------------------------

The **Identity Audit Engine** generates deterministic audit records
for identity-related operations within the WhyceID system.

The engine provides:

• identity activity traceability
• compliance audit records
• governance oversight visibility
• security monitoring signals
• forensic investigation support

Audit records are analytical outputs only.

The engine does not mutate domain aggregates.

Audit records may later be anchored into **WhyceChain** as evidence
by the runtime governance pipeline.

ENGINE EXECUTION
------------------------------------------------------------

The engine performs the following steps:

1. Validate audit command
2. Generate deterministic AuditId
3. Build immutable audit record
4. Produce audit output

AuditId must be generated using SHA256:

SHA256(identityId + action + timestamp)
