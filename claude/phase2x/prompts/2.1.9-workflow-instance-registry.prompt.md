COMPONENT PURPOSE
------------------------------------------------------------

The Workflow Instance Registry tracks runtime workflow executions within the WSS system.

Each time a workflow is started, an instance is created and registered.

The registry maintains metadata for:

- workflow instances
- workflow execution status
- correlation identifiers
- execution timestamps
- instance ownership

The registry allows the runtime dispatcher and event router to locate and manage active workflows.
