# Component 2.2.27 — Capital Balance Engine

## System Category
Capital System

## Architecture Layer
Engine Layer — T3I Intelligence Engine

## Purpose
Computes real-time capital balance snapshots across pools, investors, reservations, allocations, and utilizations within the Whycespace economic infrastructure.

## Key Properties
- Stateless, deterministic, thread-safe
- No state mutation, no event emission
- Read-side analytical calculations only

## Input
ComputeCapitalBalanceCommand: PoolId, InvestorIdentityId (optional), Currency, RequestedBy, RequestedAt

## Output
CapitalBalanceSnapshot: TotalCommittedCapital, TotalContributedCapital, TotalReservedCapital, TotalAllocatedCapital, TotalUtilizedCapital, TotalDistributedCapital, AvailableCapital

## AvailableCapital Formula
TotalContributedCapital - TotalReservedCapital - TotalAllocatedCapital - TotalUtilizedCapital

## Data Sources
- Capital Registry (ICapitalRegistry)
- Capital Records (CapitalRecord)
