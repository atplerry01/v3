# 2.0.67 — Governance Evidence Recorder

## Component
- **Name**: Governance Evidence Recorder
- **Category**: Governance Adapter
- **Tier**: T0U — Constitutional Engine
- **Layer**: Engine Layer

## Location
`src/engines/T0U/Governance/GovernanceEvidenceRecorder.cs`

## Purpose
Records governance events as cryptographic evidence anchored to WhyceChain.
Ensures governance actions are immutable, verifiable, and auditable.

## Execution Flow
```
RecordGovernanceEvidenceCommand
  → Validate (proposalId, evidenceType, eventRef, payload, guardianId)
  → Compute SHA-256 evidence hash (canonical JSON)
  → Compute Merkle root from evidence leaves
  → Submit evidence to WhyceChain via ChainEvidenceGateway
  → Return GovernanceEvidenceResult
  → GovernanceEvidenceRecordedEvent (emitted by runtime)
```

## EvidenceType Enum
- ProposalCreated
- VoteCast
- QuorumEvaluated
- DecisionMade
- DisputeRaised
- EmergencyAction
- WorkflowTransition

## Dependencies
- `ChainEvidenceGateway` (T0U WhyceChain)
- `System.Security.Cryptography` (SHA-256)

## Validation Rules
- Proposal ID must not be empty
- Event reference ID must not be empty
- Evidence type must be valid enum value
- Evidence payload must not be empty/whitespace
- Guardian ID must not be empty
