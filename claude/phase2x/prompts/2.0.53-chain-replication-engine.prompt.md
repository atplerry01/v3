## Description

Stateless, deterministic engine that prepares replication payloads for WhyceChain state synchronization.
Selects blocks and ledger entries up to a target height and computes a SHA256 replication hash
over replicated height, entry count, and latest block hash.

Does NOT perform network replication, persistence, or chain mutation.
