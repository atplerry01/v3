# TRUSTSCORE ENGINE

This prompt must be executed using the Whycespace MASTER PROMPT GENERATION TEMPLATE.

This component implements the TrustScore Engine responsible for calculating identity trust scores.

TrustScore is a computed value representing the trustworthiness of an identity based on:

verification status
identity attributes
account activity
device trust
governance behavior
economic participation

TrustScore is an analytical evaluation and therefore belongs to the Intelligence Engine tier.

claude/bootstrap-prompts/master-bootstrap.prompt.md

must be respected.

SECTION 6 — TRUST SCORE CALCULATION
------------------------------------------------------------

TrustScore must be computed using weighted scoring.

Example weights:

Email Verification: +10

Phone Verification: +10

Document Verification: +20

Device Trust: +15

Account Age: +10

Behavior Score: +20

Maximum score: 100

TrustScore must be normalized between 0 and 100.
