# 2.1.25 WORKFORCE PERFORMANCE ENGINE

You are implementing the **Workforce Performance Engine**.

This engine evaluates **worker performance metrics** across completed assignments and scheduled tasks within the HEOS system.

Performance evaluation supports:

• workforce productivity analysis  
• incentive calculation  
• operational optimization  
• workforce governance  

Performance results will be used by:

• Workforce Incentive Engine  
• Workforce Compliance Engine  
• Workforce Lifecycle Engine  

The engine operates on:

WorkforceAggregate

Domain models exist in:

src/domain/core/workforce/

This implementation must follow **WBSM v3 Prompt Structural Rules and Guardrails**.

# STEP 7 — PERFORMANCE RULES

Performance score should be calculated using weighted factors:

TaskCompletionRate  
AverageTaskDuration  
CustomerRating  

Example scoring concept:

Score = weighted calculation of efficiency and quality.

Incorrect:

WorkforcePerformanceEngine → another engine
