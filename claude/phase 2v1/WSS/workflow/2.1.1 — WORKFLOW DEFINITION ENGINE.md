# WHYCESPACE WBSM v3

# PHASE 2.1.1 — WORKFLOW DEFINITION ENGINE

You are implementing the Workflow Definition Engine for the Whycespace Workflow Structural System (WSS).

This engine defines workflows used by the runtime.

IMPORTANT RULES

• Engines are stateless  
• Engines never call engines  
• Execution is handled by WorkflowRuntime
