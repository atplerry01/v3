# WHYCESPACE WBSM v3
# PHASE 2.1.6 — WORKFLOW VALIDATION ENGINE

You are implementing the Workflow Validation Engine for the Whyce Structural System (WSS).

This engine performs centralized validation of workflows before they are accepted by the system.

It orchestrates validation across workflow definition, graph structure, template resolution, and version integrity.

---

# ARCHITECTURE RULES

Follow WBSM v3 strictly.

• Engines are stateless  
• Engines never call engines directly in execution flow  
• Validation must occur before workflow registration  
• Validation must be deterministic  
• Validation must produce structured validation results
