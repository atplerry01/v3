# ENGINE IMPLEMENTATION

Create:

WorkflowLifecycleEngine.cs

Responsibilities:

StartWorkflow

• retrieve workflow definition from registry  
• create workflow instance via instance registry  
• initialize workflow state  
• determine first step via WorkflowGraphEngine  
• emit WorkflowStarted event  
• return LifecycleDecision  

---

AdvanceStep

• determine next step using WorkflowGraphEngine  
• update workflow state store  
• emit WorkflowStepStarted event  
• return LifecycleDecision  

---

CompleteStep

• record step completion in WorkflowStateStore  
• clear step timeout using WorkflowTimeoutEngine  
• emit WorkflowStepCompleted event  
• advance to next step using WorkflowGraphEngine  

---

FailStep

• retrieve retry count  
• evaluate retry policy using WorkflowRetryPolicyEngine  

If retry allowed:

• return LifecycleDecision indicating retry step  

If retry exhausted:

• mark workflow state as Failed  
• emit WorkflowStepFailed event  

---

CompleteWorkflow

• update workflow status to Completed  
• emit WorkflowCompleted event  

---

TerminateWorkflow

• update workflow status to Cancelled  
• emit WorkflowCancelled event
