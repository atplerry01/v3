# WHYCESPACE WBSM v3
# PHASE 2.1.8 — WORKFLOW STEP ENGINE MAPPING

You are implementing the Workflow Step Engine Mapping for the Whyce Structural System (WSS).

This engine maps workflow steps to execution engines.

Workflow steps reference engines by name, and this mapping resolves those references to the runtime engine identifiers used by the execution layer.

This mapping enables the workflow runtime to dispatch steps to the correct execution engine.

---

# ARCHITECTURE RULES

Follow WBSM v3 strictly.

• Engines are stateless  
• Engines never call engines directly  
• Runtime dispatches execution engines  
• Mapping must be deterministic  
• Step → Engine resolution must be fast
