---

# STEP 7 — LIFECYCLE TRANSITION RULES

The engine must enforce valid transitions.

Allowed transitions:

Registered → Active  
Active → Unavailable  
Active → Suspended  
Unavailable → Active  
Suspended → Active  
Active → Terminated  

Forbidden transitions:

Terminated → Active  
Suspended → Registered  
Unavailable → Registered  

Termination is final.

Reactivation is only allowed from Suspended.

Invalid transitions must produce a rejected decision.
