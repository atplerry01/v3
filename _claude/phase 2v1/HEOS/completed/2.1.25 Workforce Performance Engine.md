# STEP 7 — PERFORMANCE RULES

The engine must enforce:

CompletedTasks must be ≥ 0

FailedTasks must be ≥ 0

CustomerRating must be between 0 and 5

EvaluationPeriodEnd must be later than EvaluationPeriodStart

Evaluation period must not be zero duration

Performance score should use weighted factors:

TaskCompletionRate  
AverageTaskDuration  
CustomerRating  

Example scoring model:

CompletionRateWeight  
EfficiencyWeight  
QualityWeight  

Score = deterministic weighted calculation.

Tier classification must follow bounded thresholds.

Example concept:

Exceptional → top tier  
High → above standard performance  
Standard → acceptable performance  
Low → below acceptable performance
