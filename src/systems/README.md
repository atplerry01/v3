# Whycespace Systems Layer

This directory contains the implementations of the core Whycespace systems.

Systems represent macro-level operational subsystems of the ecosystem.

## Categories

### Upstream Systems
- WhyceID
- WhycePolicy
- WhyceChain
- Governance

### Midstream Systems
- WSS
- HEOS
- WhyceAtlas
- WhycePlus
- Capital

### Downstream Systems
- Cluster systems
- SPV systems
- Brand systems

Systems orchestrate domain and engines but do not contain execution engines.

Business logic remains in `/engines`.
